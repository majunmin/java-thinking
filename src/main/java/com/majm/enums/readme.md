## EnumSet

类似于 bitSet 结构

java 中一个 int 类型32位, 可以表示 32 个 值(一个bit 表示一个)
如果 要表示 N 个 值， 需要 n = N / 32 个 int 数，
m = N % 32 == (N & (32-1))  , 

第n个int值的 第m个比特位就是 要表示的那个值.
